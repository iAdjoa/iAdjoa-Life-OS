# 🌟 My Personal Life App

A custom-built dashboard to track my daily habits, moods, and goals. This app uses a **static frontend** hosted on GitHub Pages and **Google Sheets** as a lightweight database.

## 🚀 Live Demo
You can access my app here: 
👉 [https://github.io](https://github.io)

## 🛠️ Features
- **Habit Tracking:** Log daily wins and streaks.
- **Mood Journal:** A quick way to record how the day went.
- **Data Sync:** All inputs are sent directly to a private Google Sheet.
- **Mobile Friendly:** Designed to be used as a shortcut on my phone's home screen.

## 🏗️ How it Works
1. **Frontend:** Simple HTML5, CSS3, and Vanilla JavaScript.
2. **Database:** Google Sheets acts as the "backend" via a Google Apps Script API.
3. **Hosting:** Automatically deployed via **GitHub Pages**.

## 🔒 Privacy Note
Since this repository is public, I do **not** store any sensitive API keys or personal passwords directly in the code. Personal data is kept secure within my private Google account.
